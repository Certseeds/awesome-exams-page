---
title: 如何引用
---

如果你的论文、报告、课程项目或数据整理工作使用了本仓库，请至少给出一条仓库级引用。

当前推荐把本仓库视为一个数据仓库进行引用；如果后续发布了 DOI，则应优先引用 DOI 对应的正式发布版本。

## 最低引用要求

1. 引用仓库本身，并注明版本号、提交哈希或访问日期。
2. 如果实际使用了某一份试卷、答案或 notebook，请在正文、脚注或附录中补充具体页面路径。
3. 如果使用内容同时来自本仓库继承的上游 Awesome-Exams，建议同时说明本仓库为整理/衍生来源，并在需要时补充上游项目引用。

## 建议写法

可按下面形式组织参考文献：

Certseeds, & NikeTacoHub. awesome-exams-page. GitHub repository. Version 0.0.1. [https://github.com/Certseeds/awesome-exams-page](https://github.com/Certseeds/awesome-exams-page)

如果你引用的是站点中的具体页面，建议再补一条文内说明，例如：

使用的数据来自 awesome-exams-page 中的 MA103-线性代数课程页面及其对应 Markdown / Jupyter notebook。

## BibTeX 示例

```bibtex
@misc{certseeds_awesome_exams_page,
  author       = {Certseeds and NikeTacoHub},
  title        = {awesome-exams-page},
  howpublished = {\url{https://github.com/Certseeds/awesome-exams-page}},
  note         = {Version 0.0.1; accessed 2026-05-28}
}
```

## 引用与许可证

引用并不替代许可证义务。

仓库中的试题整理内容、页面内容等遵循 CC BY-NC-SA 4.0；仓库中的代码部分遵循 AGPL-v3.0-or-later。若你直接复制题面、答案、notebook 内容或代码，还需要分别满足对应许可证的署名、非商用、共享方式或源码开放等要求。
