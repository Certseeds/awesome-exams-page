### 6. 级数分析

$$
\frac{a}{\sqrt{(1+a)(1+b)}}+
\frac{b^{2}}{\sqrt{(1+a^{2})(1+b^{2})}}+
\frac{a^{3}}{\sqrt{(1+a^{3})(1+b^{3})}}+
\frac{b^{4}}{\sqrt{(1+a^{4})(1+b^{4})}}+
\cdots+
\frac{c_{n}}{\sqrt{(1+a^{n})(1+b^{n})}}+
\cdots
$$

where

$$
C_{n}= \begin{cases}a^{n}, & \text { if } n \text { is odd} \\ b^{n}, & \text { if } n \text { is even}\end{cases}
